# MarioKartAdvance Font

This font is a fan recreation of the typeface used in Nintendo's Mario Kart Advance software for the Game Boy Advance (AGB-AMKJ).

## Disclaimer

This is NOT an official Nintendo product. All original designs are Â© Nintendo / Intelligent Systems

This font was created by tracing the original pixel glyphs from the Mario Kart Advance ROM for use in fan preservation and restoration projects.

## Usage

This recreation is provided free for personal and non-commercial use. For commercial use, please obtain appropriate permissions from the original rights holders.

## Technical Details

- Native size: 8x16px
- Character set: Covers Latin alphabet, numbers, and common punctuation as present in the original software
- Formats: TTF
