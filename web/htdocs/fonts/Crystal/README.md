# pokecrystal Font

This font is a fan recreation of the typeface used in Nintendo's Pokémon Crystal software for the Game Boy Color (all languages).

## Disclaimer

This is NOT an official Nintendo product. All original designs are © Nintendo / The Pokémon Company / Creatures Inc.

This font was created by tracing the original pixel glyphs from the Pokémon Crystal ROM for use in fan preservation and restoration projects.

## Usage

This recreation is provided free for personal and non-commercial use. For commercial use, please obtain appropriate permissions from the original rights holders.

## Technical Details

- Native size: 8px
- Character set: Covers Latin alphabet, numbers, and common punctuation as present in the original software
- Formats: TTF
